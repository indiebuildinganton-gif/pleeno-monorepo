# Entities Domain Migrations

This folder will contain migrations for entities, clients, and representatives.

## Purpose
Manages the core business entities that agencies track and manage.

## Planned Tables
- `entities` - Business entities (individuals, companies, partnerships, trusts)
- `clients` - Clients that entities represent
- `representatives` - Representatives/contacts for entities
- `entity_relationships` - Relationships between entities
